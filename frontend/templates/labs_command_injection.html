Lab Briefing

In many applications, network tools like ping are executed by calling shell commands. If input values are simply concatenated into a system shell prompt, special operators like &, |, or ; can be injected to run arbitrary code on the hosting server.

Objective:
  1. Inject a command using characters like & or | in Vulnerable Mode (e.g. 127.0.0.1 && whoami).
  2. Verify that the status changes to EXPLOITED.
  3. Toggle to Secure Mode and check that the same payload is blocked. Once verified, the lab becomes SOLVED!
